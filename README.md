# Spring-Boot-Project---Book-Management
Book Project is created on various levels →
Level 1. Adding Manual Data into List & Rest API’s testing on POSTMAN.
Level 2. With This Manual Data, Using ResponseEntity & testing on POSTMAN.
Level 3. Connecting with Database, using JPA methods, CRUD operations on Postgres – Testing on POSTMAN.
Level 4. Creating Uni-Relational-Mappings between Book & Author Entity – Testing on POSTMAN.
Level 5. Creating Bi-Relational-Mappings between Book & Author Entity with the help of Jackson – Testing on POSTMAN.
